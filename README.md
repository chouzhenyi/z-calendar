# z-calendar

## 一个PC的日历插件 基于vue2.x
```
通过vue-cli@3.0 构建的项目；日历插件核心部分是calendar.js ，主要逻辑都在这里；
用到的道友可以根据自己的需要自行扩展和改写
```

### 启动方式
```
npm install
npm run serve
```
### 关于参数

没有什么参数是b必须的，但可以根据需要配置中英文、是否展示时间控件（分钟数的间隔）、是否展示当天之前的时间等

demo https://chouzhenyi.github.io/z-calendar/
